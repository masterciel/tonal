```js
var scaleChords = require('scale-chords')
var chords = scaleChords('C major', 4)
chords('II') // => ['D', 'F', 'A', 'C']
```

This is part of [tonal](https://www.npmjs.com/package/tonal)

## Installation

Via npm: `npm i --save scale-chords`
