pcset

Functions to create and manipulate pitch class sets

Example
var pcset = require('tonal-pcset')
pcset.equal('c2 d5 e6', 'c6 e3 d1') // => true

Methods

(static) chroma(set) → {String}

Get chroma of a pitch class set. A chroma identifies each set uniquely. It's a 12-digit binary each presenting one semitone of the octave.

Note that this function accepts a chroma as parameter and return it without modification.

Parameters:
Name Type Description
set Array | String

the pitch class set
Returns:

a binary representation of the pitch class set

Type
String
Example
pcset.chroma('C D E') // => '1010100000000'

(static) chromaModes(set, normalize) → {Array.<String>}

Given a pitch class set (a list of notes or a pitch class set chroma), produce the 12 rotations of the chroma (and discard the ones that starts with '0')

This can be used, for example, to get all the modes of a scale.

Parameters:
Name Type Description
set Array | String

the list of notes or pitchChr of the set
normalize Boolean

(Optional, true by default) remove all the rotations that starts with '0'
Returns:

an array with all the modes of the chroma

Type
Array.<String>
Example
pcset.chromaModes('C E G')

(static) filter(set, notes) → {Array}

Filter a list with a pitch class set

Parameters:
Name Type Description
set Array | String

the pitch class set notes
notes Array | String

the note list to be filtered
Returns:

the filtered notes

Type
Array
Example
pcset.filter('c d e', 'c2 c#2 d2 c3 c#3 d3') // => [ 'c2', 'd2', 'c3', 'd3' ])
pcset.filter('c2', 'c2 c#2 d2 c3 c#3 d3') // => [ 'c2', 'c3' ])

(static) fromChroma(binary, tonic) → {Array}

Given a pitch class set in binary notation it returns the intervals or notes (depending on the tonic)

Parameters:
Name Type Description
binary String

the pitch class set in binary representation

Returns:

a list of notes or intervals

Type
Array
Example
