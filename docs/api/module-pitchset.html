pitchset

Functions to create and manipulate pitch sets

Example
var pitchset = require('tonal-pitchset')
pitchset.equal('c2 d5 e6', 'c6 e3 d1') // => true

Methods

(static) chroma(set) → {String}

Get chroma of a pitch set. A chroma identifies each pitch set uniquely. It's a 12-digit binary each presenting one semitone of the octave.

Note that this function accepts a chroma as parameter and return it without modification.

Parameters:
Name Type Description
set Array | String

the pitch set
Returns:

a binary representation of the pitch set

Type
String
Example
pitchset.chroma('C D E') // => '1010100000000'

(static) chromaModes(set, normalize) → {Array.<String>}

Given a pitch set (a list of notes or a pitch set chroma), produce the 12 rotations of the chroma (and discard the ones that starts with '0')

This can be used, for example, to get all the modes of a scale.

Parameters:
Name Type Description
set Array | String

the list of notes or pitchChr of the set
normalize Boolean

(Optional, true by default) remove all the rotations that starts with '0'
Returns:

an array with all the modes of the chroma

Type
Array.<String>
Example
pitchset.chromaModes('C E G')

(static) fromChroma(binary, tonic) → {Array}

Given a pitch set in binary notation it returns the intervals or notes (depending on the tonic)

Parameters:
Name Type Description
binary String

the pitch set in binary representation

the pitch set tonic
Returns:

a list of notes or intervals

Type
Array
Example
