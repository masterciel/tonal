pitchset

Functions to create and manipulate pitch sets

Example
var pitchset = require('tonal-pitchset')

Methods

(static) notes(notes) → {Array.<String>}

Get the notes of a pitch set. The notes in the set are sorted in asceding pitch order, and no repetitions are allowed.

Note that it creates pitch sets and NOT picth class sets. This functionallity resides inside tonal-pcset module.

Parameters:
Name Type Description
notes String | Array

the notes to create the pitch set from
Returns:

the ordered pitch set notes

Type
Array.<String>
Example
pitchset.notes('C4 c3 C5 c4') // => ['C3', 'C4', 'C5']
