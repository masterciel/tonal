progression

Work with chord progressions.

Example
var progression = require('tonal-progression')
progression.abstract('Cmaj7 Dm7 G7', 'C')

Methods

(static) abstract(chords, tonic) → {Array}

Given a chord progression and a tonic, return the chord progression with roman numeral chords.

Parameters:
Name Type Description
chords Array | String

the chord progression
tonic String

the tonic
Returns:

the chord progression in roman numerals

Type
Array
Example
progression.abstract('Cmaj7 Dm7 G7', 'C') // => [ 'Imaj7', 'IIm7', 'V7' ]

(static) parseRomanChord(str) → {Object}

Parse a chord expressed with roman numerals. It returns an interval representing the root of the chord relative to the key tonic and the chord name.

Parameters:
Name Type Description
str String

the roman numeral string
Returns:

the roman number in array notation or null if not valid numeral

Type
Object
Example
var parse = require('music-notation/roman.parse')
parse('V7') // => { root: ['tnl', 1, 0, 0, 1], name: '7'}
parse('bIIalt') // => [ root: ['tnl', -5, 0, 2, 1], name: 'alt']

(static) romanRegex() → {RegExp}

Returns a regex to match roman numbers literals with the from: [accidentals]roman[element].

The executed regex contains:

  • input: the input string
  • accidentals: (Optional) one or two flats (b) or shaprs (#)
  • roman: (Required) a roman numeral from I to VII either in upper or lower case
  • element: (Optional) a name of an element
Returns:

the regexp

Type
RegExp
Example
var r = progression.romanRegex()
r.exec('bVImaj7') // => ['bVImaj7', 'b', 'VI', 'maj7'])
r.exec('III dom') // => ['III dom', '', 'III', 'dom'])
