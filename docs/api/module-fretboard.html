(static) simpleTuning(name) → {Array}

Given a tuning name returns the notes of the strings in open position as pitch classes removing doubled strings.

Parameters:
Name Type Description
name String

the tuning name or notes of the strings in open position
Returns:

the string notes as pitch classes

Type
Array
Example
fret.simpleTuning('guitar') => [ 'E', 'A', 'D', 'G', 'B', 'E' ]
fret.simpleTuning('charango') => [ 'G', 'C', 'E', 'A', 'E' ]

Parameters:
Name Type Description

the tuning name
Returns:

the string notes or null if not valid tuning name

Type
Array
Example

fret.tuning('guitar') // => [ 'E2', 'A2', 'D3', 'G3', 'B3', 'E4' ]
