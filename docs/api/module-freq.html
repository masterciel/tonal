freq

A collection of modules to work with note frequencies

Example
var freq = require('tonal-freq')
freq.toFreq('A4') // => 440
freq.note(440) // => 'A4'
freq.noteAndDetune(320) // => ['C4', 200]

Methods

(static) cents(base, freq) → {Integer}

Get difference in cents between two frequencies. The frequencies can be expressed with hertzs or midi numbers or note names

Parameters:
Name Type Description
base Float | Integer | String
freq Float | Integer | String
Returns:

The difference in cents

Type
Integer
Example
import { cents } from 'tonal-freq'
cents('C4', 261) // => -4

(static) eqTempFreq(ref, maxDecimals, note) → {Number}

Return the equal tempered frequency of a note.

This function can be partially applied if note parameter is not present.

Parameters:
Name Type Description
ref Float

the tuning reference
maxDecimals Integer

(Optional) the maximum number of decimals (all by default)
note String | Pitch

the note to get the frequency from
Returns:

the frequency

Type
Number
Example
eqTempFreq(444, 4, 'C3')
const toFreq = eqTempFreq(444, 2)
toFreq('A3') // => 222
