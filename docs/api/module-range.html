range

A collection of functions to create note ranges.

Example
var range = require('tonal-range')
// ascending chromatic range
range.chromatic(['C4', 'E4']) // => ['C4', 'Db4', 'D4', 'Eb4', 'E4']
// descending chromatic range
range.chromatic(['E4', 'C4']) // => ['E4', 'Eb4', 'D4', 'Db4', 'C4']
// combining ascending and descending in complex ranges
range.chromatic(['C2', 'E2', 'D2']) // => ['C2', 'Db2', 'D2', 'Eb2', 'E2', 'Eb2', 'D2']
// numeric (midi note numbers) range
range.numeric('C4 E4 Bb3') // => [60, 61, 62, 63, 64]
// complex numeric range
range.numeric('C4 E4 Bb3') // => [60, 61, 62, 63, 64, 63, 62, 61, 60, 59, 58]
// create a scale range
range.pitchSet('c e g a', 'c2 c3 c2') // => [ 'C2', 'E2', 'G2', 'A2', 'C3', 'A2', 'G2', 'E2', 'C2' ] *
 g

Methods

(static) chromatic(list) → {Array}

Create a range of chromatic notes. The altered notes will use flats.

Parameters:
Name Type Description
list String | Array

the list of notes or midi note numbers
Returns:

an array of note names

Type
Array
Example
tonal.chromatic('C2 E2 D2') // => ['C2', 'Db2', 'D2', 'Eb2', 'E2', 'Eb2', 'D2']
// with sharps
tonal.chromatic('C2 C3', true) // => [ 'C2', 'C#2', 'D2', 'D#2', 'E2', 'F2', 'F#2', 'G2', 'G#2', 'A2', 'A#2', 'B2', 'C3' ]

(static) numeric(list) → {Array}

Create a numeric range. You supply a list of notes or numbers and it will be conected to create complex ranges.

Parameters:
Name Type Description
list String | Array

the list of notes or numbers used
Returns:

an array of numbers or empty array if not vald parameters

Type
Array
Example
range.numeric('C5 C4') // => [ 72, 71, 70, 69, 68, 67, 66, 65, 64, 63, 62, 61, 60 ]
// it works with numbers
range.numeric([10, 5]) // => [ 10, 9, 8, 7, 6, 5 ]
// complex range
range.numeric('C4 E4 Bb3') // => [60, 61, 62, 63, 64, 63, 62, 61, 60, 59, 58]
// can be expressed with a string or array
range.numeric('C2 C4 C2') === range.numeric(['C2', 'C4', 'C2'])

(static) pitchSet(scale, range) → {Array}

Create a pitch set (scale or chord) range. Given a pitch set (a collection of pitch classes), and a range array, it returns a range in notes.

Parameters:
Name Type Description
scale String | Array | function

the scale to use or a function to convert from midi numbers to note names
range String | Array

a list of notes or midi numbers
Returns:

the scale range, an empty array if not valid source or null if not valid start or end

Type
Array
Example
range.pitchSet('C D E F G A B', ['C3', 'C2'])
// => [ 'C3', 'B2', 'A2', 'G2', 'F2', 'E2', 'D2', 'C2' ]
