range

A collection of functions to create note ranges.

Example
var tonal = require('tonal')
// create numeric (midi note numbers) range
tonal.range('C4 E4 Bb3') // => [60, 61, 62, 63, 64]
// create a range with three points
tonal.range('C4 E4 Bb3') // => [60, 61, 62, 63, 64, 63, 62, 61, 60, 59, 58]
// create a chromatic range with three points
tonal.chromatic('C2 E2 D2') // => ['C2', 'Db2', 'D2', 'Eb2', 'E2', 'Eb2', 'D2']
// create a scale range
tonal.scaleRange('C E G A', 'c2 c3 c2') // => [ 'C2', 'E2', 'G2', 'A2', 'C3', 'A2', 'G2', 'E2', 'C2' ] *

(static) range(list) → {Array}

Create a numeric range. You supply a list of notes or numbers and it will be conected to create complex ranges.

Parameters:
Name Type Description
list String | Array

the list of notes or numbers used
Returns:

an array of numbers or empty array if not vald parameters

Type
Array
Example
import { range } from 'tonal-range'
range('C5 C4') // => [ 72, 71, 70, 69, 68, 67, 66, 65, 64, 63, 62, 61, 60 ]
// it works with numbers
range([10, 5]) // => [ 10, 9, 8, 7, 6, 5 ]
// complex range
range('C4 E4 Bb3') // => [60, 61, 62, 63, 64, 63, 62, 61, 60, 59, 58]
// can be expressed with a string or array
range('C2 C4 C2') === range(['C2', 'C4', 'C2'])
// included in tonal package
tonal.range('C2 C3')

(static) scaleRange(scale, range) → {Array}

Create a scale range. Given a pitch set (a collection of pitch classes), and a start and end it returns a note range.

Parameters:
Name Type Description
scale String | Array | function

the scale to use or a function to convert from midi numbers to note names
range String | Array

a list of notes or midi numbers
Returns:

the scale range, an empty array if not valid source or null if not valid start or end

Type
Array
Example
var tonal = require('tonal')
tonal.scaleRange('C D E F G A B', 'C3 C2')
// => [ 'C3', 'B2', 'A2', 'G2', 'F2', 'E2', 'D2', 'C2' ]
