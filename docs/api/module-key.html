key

Key refers to the tonal system based on the major and minor scales. This is is the most common tonal system, but tonality can be present in music based in other scales or concepts.

This is a collection of functions related to keys.

Example
var key = require('tonal-key')
key.scale('E mixolydian') // => [ 'E', 'F#', 'G#', 'A', 'B', 'C#', 'D' ]
key.relative('minor', 'C major') // => 'A minor'

(static) modes(alias) → {Array}

Get a list of valid mode names. The list of modes will be always in increasing order (ionian to locrian)

Parameters:
Name Type Description

true to get aliases names
Returns:

an array of strings

Type
Array
Example
key.modes() // => [ 'ionian', 'dorian', 'phrygian', 'lydian',
// 'mixolydian', 'aeolian', 'locrian' ]
key.modes(true) // => [ 'ionian', 'dorian', 'phrygian', 'lydian',
// 'mixolydian', 'aeolian', 'locrian', 'major', 'minor' ]

(static) relative(mode, key)

Get relative of a key. Two keys are relative when the have the same key signature (for example C major and A minor)

It can be partially applied.

Parameters:
Name Type Description
mode String

the relative destination
key String

the key source
Example
key.relative('dorian', 'B major') // => 'C# dorian'
// partial application
var minor = key.relative('minor')
minor('C major') // => 'A minor'
minor('E major') // => 'C# minor'
