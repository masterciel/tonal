key

A collection of functions related to music keys. Things like keys signatures, scales, modes, etc.

Example
var key = require('tonal-key')
key.scale('E mixolydian') // => [ 'E', 'F#', 'G#', 'A', 'B', 'C#', 'D' ]
key.relative('minor', 'C major') // => ['minor', 'A']
